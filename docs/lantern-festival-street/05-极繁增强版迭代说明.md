# 05. 极繁增强版迭代说明（v2）

本次迭代在原有“无尽花街 + 2.5D 漫游”基础上，重点完成三件事：

1. 极繁度增强
- 增加更多街景类型：牌楼、幡旗、围观人群、更多摊位变体；
- 摊位类型拆分为灯摊、元宵摊、糖画/甜品摊、茶饮摊、面具摊；
- 动作密度提升：同一段可能并存主表演 + 围观 + 路侧摊位；
- 强化前景层：近景大灯笼和粒子增强“身临其境”感。

2. 电影感夜游调色
- 画面加入暖橙和冷蓝的双层色彩分级（color grade）；
- 增加暗角（vignette）与颗粒（film grain）气氛；
- 速度变化影响镜头脉冲和轻微摇摆，形成“镜头感”。

3. 工程接入与上线准备
- 保持原有路径：`/lantern-street/`；
- 继续使用静态构建，不引入重型依赖；
- 兼容桌面滚轮与移动端上下滑动；
- 已通过 `npm run build` 构建验证。

## 保留并强化的七类民俗动作
- 猜灯谜
- 赏花灯
- 舞龙
- 舞狮子
- 踩高跷
- 扭秧歌
- 吃元宵

## 可继续增强的方向（下一轮）
- 增加“节拍驱动”鼓点层，让舞龙舞狮节奏更明显；
- 增加“段落主题切换”（如宫灯段、谜语段、烟火段）；
- 增加可选“静谧模式”（降低密度，适合长时间观看）。
