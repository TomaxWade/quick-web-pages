# 03. 文案系统与版式规范

## 1) 文案总原则

本项目文案应服务沉浸，不打断场景：
- 少而精：不做大段说明文；
- 强意象：强调“灯海、花街、人间烟火”；
- 可引导：告诉用户如何走动和观看；
- 节庆语气：热闹但不吵闹，雅致中带民间活力。

## 2) 页面文案层级

## 2.1 一级文案（主标题）
建议候选：
1. 灯海长街 · 元宵夜行
2. 万灯如昼 · 花街无尽
3. 上元花街 · 一路生辉

使用建议：
- 字重偏重（700+）
- 书法感或古典宋黑混排
- 占据首屏视觉焦点，但不遮挡主街透视

## 2.2 二级文案（副标题）
用于明确体验指向：
- 滚动鼠标，沿花街向前或向后漫游。
- 猜灯谜、赏花灯、舞龙舞狮，在灯火中轮番上演。

## 2.3 操作提示文案
桌面端：
- 滚轮前行 / 回退
移动端：
- 上下滑动，在花街中漫游

## 2.4 场景标签文案（可选，淡入淡出）
- 谜灯巷口
- 龙舞正酣
- 狮跃鼓鸣
- 元宵暖摊
- 秧歌转街

## 3) 字体策略

建议字体配对：
- 标题：`Ma Shan Zheng` 或 `ZCOOL XiaoWei`
- 正文：`Noto Serif SC`（说明） + `Noto Sans SC`（小标签）

目标效果：
- 标题有节庆题匾感；
- 正文有阅读舒适度；
- 不使用系统默认黑体直出，避免“模板页”气质。

## 4) 色彩与视觉语义

主色板（建议）：
- 朱红：#8f1b1f
- 金箔：#f7c45c
- 琥珀光：#ffb347
- 靛夜：#101529
- 青玉：#2f7f78
- 暖白：#fff0d2

应用规则：
- 背景深色承托灯光；
- 文案以暖白为主，避免纯白刺眼；
- 重要提示用金箔色；
- 少量青玉用于冷暖平衡。

## 5) 版式布局规范

## 5.1 基础布局
- 全屏沉浸画布为主；
- 左上角/中上方叠加标题与副文案；
- 右下角轻量状态条（速度/街段提示）；
- 文案层使用半透明暗底，保持可读。

## 5.2 间距与层级
- 标题与副标题间距：12-16px
- 文案容器内边距：18-24px
- 角标与边缘留白：16-20px

## 5.3 响应式策略
- 手机端标题缩到 26-34px，副文案 14-16px；
- 隐藏非必要说明，保留操作提示与节庆核心文案；
- 保证不遮挡道路中央透视线。

## 6) 动效文案配合

文案出现节奏：
- 首屏：主标题 0.3s 渐显；
- 1.2s 后出现副标题；
- 操作提示常驻但透明度低，鼠标滚动后再淡出至 60%。

避免问题：
- 不做高频闪动字幕；
- 不做大面积跑马灯影响观看。

## 7) 可访问性与可读性

- 文字与背景对比度 >= 4.5:1；
- 关键提示不只靠颜色区分；
- 支持 `prefers-reduced-motion`，降低文案动画频率；
- 所有提示语言简洁，减少理解负担。

## 8) 文案完整性清单（发布前）

- [ ] 主标题明确“元宵/花街”主题
- [ ] 副标题说明沉浸内容（看灯 + 民俗）
- [ ] 操作提示明确滚轮行为
- [ ] 至少 3 处节庆语义词（灯、街、元宵、舞龙等）
- [ ] 手机端仍可清晰识别操作方式

该规范用于保证页面即便做极繁场景，也不会因为文案混乱而破坏沉浸。
